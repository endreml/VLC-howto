VLC keybinds
  • Toolbars: [CTRL] + [H]
  • Preferences: [CTRL] + [P]
  • Playlists/library: [CTRL] + [L]
  • Open file: [CTRL] + [O]
  • Play/pause video: [SPACE]
  • Volume: [CTRL] + [ARROW UP/DOWN]

Installation options for MacOS

  • Step 4a - A mounted folder, as pictured below, will open.

  • Step 4b - Drag VLC to applications. This saves the program to your applications folder and installs it.

Playing a video file#back to top

Step 0 - Download this video file: - examplevideo.mp4.

Step 1 - On the VLC toolbar click on "File" press “Open file…”.

Step 2 - A directory window will open, navigate to the directory of your video file.

Step 3 - Click the video file you want to view, and click "Open".

Adding subtitles to video file#back to top

Step 0 - Download this subtitle file: - examplesubtitle.srt.

Step 1 - Click the dropdown-menu [Subtitles] in the toolbar, select “Add subtitle file…”.

Step 2 - The directory where your current Vide file is located will pop up, navigate to where your subtitle file is, choose it and press “Open”.

Step 3 -The video will start playing with the subtitles.

If the subtitles and the video are not in sync then you will need to adjust the subtitle by delaying it or playing it ahead.

Step 4 - Click the dropdown-menu [Window] in the toolbar, select "Track Synchronization".

Step 5 - A small window will appear, change the values of “Subtitle track synchronization”. If the subtitle is delayed then enter the number of seconds it is delayed in negative. In this case it is delayed by 2 seconds so enter “-2” and press Enter on your keyboard as it will apply the changes.

Additional VLC function#back to top

The new functions are the video file conversion and the record function which records a part of a video you're currently watching.

Video File conversion for playback on different devices

Media devices are compatible only with file formats their manufacturers set. VLC can convert different video files to what your device is compatible with and change the resolution of the video for more efficiency when playing. This function can be found in the media tab.

VLC can record a part of a video that is playing on your screen and saves it as a .ts file which you can convert later on with the conversion function above. This function can be found in the Playback tab.

Understanding VLC#back to top

Datastructure

In order to fully utilize the program and understand it’s functions, we’ll need to look at how it is structured and how the different functions work.

Figure 2. Structure Model.

The figure above illustrates how VLC is structured. The Program accesses the files uploaded by the user from the computers directory. VLC then on some of it’s functions creates a new file and save it on the users specified directory-folder.

Functions

VLC functions can be divided into two process, read and write. The read process reads the files uploaded by the user and plays it. When a video file is played, other processes can be used to overlay a subtitle file, and various postprocess effects (ie. color grading) the video. While reading the file, synchronizing both the audio and subtitles to the video is also possible. The function processes can be seen below.
